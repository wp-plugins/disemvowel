== Description ==

Simple plugin designed to help regulate all comments. Based on a suggested idea: http://wordpress.org/extend/ideas/topic.php?id=1976

== Installation ==

1. Upload `disemvowel.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

That's it! It will automatically filter your comments.
